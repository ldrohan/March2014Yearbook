== README

General Assembly's Web Development Immersive class graduated in March 2014 in San Francisco, CA. This is our yearbook.
Site created by Taylor Chiu, Luke Drohan (github.com/ldrohan) and David Korsunsky (github.com/daveyk535)

* Ruby version 4.1.1

* System dependencies
  * We added parallax scrolling with Skrollr: https://github.com/reed/skrollr-rails
  * Jquery
  * PostgreSQL database
  
* Deployed to heroku at http://wdimarch2014.herokuapp.com/
